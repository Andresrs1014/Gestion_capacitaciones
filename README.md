# Gestion Capacitaciones

Este repositorio contiene el frontend (React) y backend (FastAPI) para la gestión de capacitaciones en IMC CARGO INTERNATIONAL SAS.

## Estructura del proyecto

- `frontend/`: Aplicación React
- `backend/`: API FastAPI y base de datos

## Configuración inicial

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/gestion_capacitaciones.git
   ```
2. Instala las dependencias en cada carpeta (`frontend` y `backend`).
3. Configura las variables de entorno necesarias.
4. Ejecuta el backend y frontend según la documentación interna.
